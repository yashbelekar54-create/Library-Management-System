## Account Registration

### New User Registration Process

1. **Access Registration Page**
   - Click "Create New Account" on the welcome page
   - Select your role (Student, Faculty, or Staff)

2. **Step 1: Basic Information**
   - Full Name
   - Email Address (institutional email preferred)
   - Phone Number
   - Choose a strong password

3. **Step 2: Role-Specific Details**

   **For Students:**
   - Student ID Number
   - Department/Program
   - Year of Study
   - Expected Graduation Date

   **For Faculty:**
   - Employee ID
   - Department
   - Designation
   - Research Interests

   **For Staff:**
   - Employee ID
   - Department
   - Job Title

4. **Step 3: Document Verification**
   - Upload ID proof (Student ID/Employee ID)
   - Upload photo
   - Upload proof of affiliation

5. **Step 4: Preferences**
   - Preferred Language
   - Notification Settings
   - Subject Interests
   - Privacy Preferences

6. **Account Approval**
   - Submit application
   - Wait for approval (usually 24-48 hours)
   - Receive confirmation email

## Borrowing and Returns

### Borrowing Process

1. **Search and Select**
   - Find desired book
   - Check availability
   - Click "Reserve" or "Borrow Now"

2. **RFID Scanning** (Self-Service)
   - Present library card to RFID reader
   - Place book on scanning pad
   - Confirm transaction
   - Receive digital receipt

3. **Manual Process** (With Librarian)
   - Present book and library card
   - Librarian scans both items
   - Confirm loan details
   - Receive receipt

### Return Process

1. **Self-Service Return**
   - Use return kiosk or drop box
   - Scan book barcode/RFID
   - Confirm return
   - System updates automatically

2. **Manual Return**
   - Present book to librarian
   - Verify return condition
   - Clear any fines if applicable

### Renewal
- Online renewal through dashboard
- Maximum 2 renewals per book
- Automatic renewal if no reservations
- Email/SMS reminders before due date

## Troubleshooting

### Common Issues

**1. Login Problems**
- **Forgot Password**: Use "Reset Password" link
- **Account Locked**: Contact librarian after 5 failed attempts
- **Email Not Verified**: Check spam folder for verification email

**2. Search Issues**
- **No Results**: Check spelling, try broader terms
- **Slow Search**: Clear browser cache, check internet connection
- **Missing Books**: Report to librarian for investigation

**3. Borrowing Problems**
- **Book Not Available**: Check if on hold, use reservation system
- **RFID Not Working**: Use manual checkout with librarian
- **Loan Limit Reached**: Return books or request extension

**4. Technical Issues**
- **App Crashes**: Update to latest version
- **Slow Loading**: Check internet connection, try refresh
- **Sync Problems**: Logout and login again
